# Template Directory

Put cropped sign templates here before the competition.

Required filenames:

- `turn_left.png`
- `turn_right.png`
- `straight.png`
- `speed_limit.png`
- `speed_unlimit.png`
- `lane_change_left.png`
- `lane_change_right.png`
- `lane_change.png`
- `danger.png`

Recommendations:

- Crop only the sign face, not the full background.
- Keep one sign per file.
- Use PNG.
- Start from `120x120` to `240x240` pixels.
- Capture templates under the same camera and lighting used in the arena.
